# KPI Prototype

## Prérequis
- Node 18+
- npm

## Installation
```bash
npm install
```

## Développement
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Déploiement (Vercel)
- Connectez le repo GitHub sur Vercel
- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Variables d'env (si besoin): `VITE_API_URL` (voir `.env.example`)
